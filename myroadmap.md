# 🗺️ My Technical Roadmap (2026)
> *Target: DevSecOps Engineer*

## 🟦 Phase 1: The Foundation (SysAdmin & Networking)
The goal is to master the operating system and understand how data moves across networks.

- [ ] **Linux Essentials:** Open Source philosophy, basic command line, and file system navigation.
- [ ] **Linux Mastery (RHCSA):** Process management, advanced permissions, LVM, and package administration.
- [ ] **Networking:** TCP/IP protocols, DNS, OSI Model, Subnetting, and Firewalls.
- [ ] **Certifications:**
    - [ ] **LPI Linux Essentials** (Entry point) 🎓 - *Currently Studying* 🐧
    - [ ] **Red Hat Certified System Administrator (RHCSA EX200)** 
    - [ ] **CompTIA Network+**

## 🟨 Phase 2: The DevOps Language (Version Control & Scripting)
The goal is to stop doing manual tasks and start treating infrastructure as code.

- [ ] **Version Control:** Advanced Git (Branching, Merging, Rebase).
- [ ] **Scripting:** Python for automation and advanced Bash scripting.
- [ ] **Tools:** GitHub, GitLab, VS Code.
- [ ] **Certification:** PCAP (Python Certified Associate Programmer).

## 🟩 Phase 3: Modern Infrastructure (Cloud & Containers)
The goal is to master where and how modern applications are executed.

- [ ] **Virtualization & Containers:** Docker, Podman.
- [ ] **Orchestration:** Kubernetes (K8s).
- [ ] **Cloud Computing:** AWS (Global Infrastructure).
- [ ] **Certifications:** - [ ] **AWS Certified Solutions Architect – Associate**
    - [ ] **CKA (Certified Kubernetes Administrator)**

## 🟧 Phase 4: Infrastructure as Code (IaC) & CI/CD
The goal is to automate the deployment of entire infrastructures with a single command.

- [ ] **Infrastructure as Code:** Terraform.
- [ ] **Configuration Management:** Ansible.
- [ ] **Pipelines:** GitHub Actions / GitLab CI.
- [ ] **Certification:** **HashiCorp Certified: Terraform Associate**.

## 🟥 Phase 5: DevSecOps Specialization (Security Shift-Left)
The goal is to "Shift Left" by integrating security from the very beginning of the lifecycle.

- [ ] **Security:** SAST/DAST analysis, Secret Management, Container Runtime security.
- [ ] **Tools:** Snyk, HashiCorp Vault, SonarQube, Trivy, Falco.
- [ ] **Certifications:**
    - [ ] **CompTIA Security+**
    - [ ] **Certified DevSecOps Professional (CDP)**
    - [ ] **CKS (Certified Kubernetes Security Specialist)**
